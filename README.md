## About
REST API for [ANS](https://ans.gg) domains metrics.

## Endpoints

### 1- Statistics

- `GET /stats`

### 2- Top Holders Leaderboard

- `GET /leaderboard`

### 3- Recent Feeds

Type: `listing` or `mints`

- `GET /recents/:type`
